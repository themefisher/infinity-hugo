## 🔑Key Features

- 📄 Onepage template
- 📊 Google Analytics support
- ⚙️ Netlify settings predefine
- 🅱️ Bootstrap Based
- ⏲️ Countdown timer available
- ✉️ Contact form support
- 🚀 Google Page Speed optimized
- 🌐 Open Graph meta tag
- 🐦 Twitter Card meta tag

## 🔧Local development

```bash
# clone the repository
git clone git@github.com:gethugothemes/infinity-hugo.git

# setup project
$ npm run project-setup

# Start local dev server
$ npm run dev
```

Or Check out [Full Documentation](https://docs.gethugothemes.com/Infinity/?ref=github).
